# Controls

* L -> Right

* J -> Left

* I -> Up

* K -> Down

* + (Numpad) -> Zoom In

* - (Numpad) -> Zoom Out

* Arrow Keys -> Move Camera

* R -> Rotate Camera

# Rules

Collect the fruit blocks around the maze to earn points, if you touch the death balls then it's game over. Object positions reset every few seconds.
